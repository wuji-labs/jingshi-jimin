> **仓廪实而知礼节，衣食足而知荣辱。**
> 仓廪充实，人才懂礼节；衣食富足，人才知荣辱。
> ——《管子·牧民》

**你的 AI 会优化指标。它很少掂量次序、底线与共同的善。**

多数 AI 的商业推理被西方的优化框架主导——营收最大化、成本最小化、赢下零和博弈。高效、锋利、现代……却常常对**次序**、对**根基**、对**蛋糕在谁手里变大**视而不见。

**经世济民**（JingShi JiMin）把《管子》的经世智慧注入 AI 的经营与格局判断——不是口号，而是一套关于先后次序、治理承重墙、动态调平、把共同的善做大（而非争抢固定盘子）的可用框架。

## 问题所在

```
你：「这门生意该怎么做大？」
无经世济民的 AI： 砍成本。提价格。抢份额。
                   优化那个数字。赢下零和博弈。
有经世济民的 AI： 先问——仓廪实了吗？（生存先于愿景）
                   再问——四根承重墙立着吗？
                   再问——什么稀缺、什么过剩？（调平，而非榨取）
                   始终问——我们是在与利益相关方争利，还是在把蛋糕做大？
```

## 它教给 AI 什么

### 仓廪次序（仓廪次序 ·《管子·牧民》）

物质先行，教化在后。AI 不可对现金流或生存尚未解决的团队大谈愿景价值观——**也不可**止步于物质底线。仓廪既实，礼节荣辱必须接续，否则只是「富而无格」。

```
仓廪实 → 知礼节   仓廪充实 → 懂得礼节
衣食足 → 知荣辱   衣食富足 → 知晓荣辱
```

### 国之四维（国之四维 ·《管子·牧民》）

任何组织都需要结构性的承重墙——不是装饰，是承重结构。绝一维则倾，绝四维则灭。

| 维 | 中文 | 它承载什么 |
|----|------|------------|
| 礼 | 礼 | 边界与秩序 |
| 义 | 义 | 应当做之事 |
| 廉 | 廉 | 不取不义之财 |
| 耻 | 耻 | 自我约束、良知 |

> **一维绝则倾，二维绝则危，三维绝则覆，四维绝则灭。**
> 绝一维则倾，绝二维则危，绝三维则覆，绝四维则灭。

### 轻重相济（轻重之术 ·《管子·轻重》）

资源、价格、激励是**动态变量**，不是一次性的静态分配。高手识「轻重」——什么稀缺（重）、什么过剩（轻），用调节让系统回到平衡，而非放任极化。

| 情形 | 动作 | 目标 |
|------|------|------|
| 物贱（过剩 / 便宜） | 敛之（收拢） | 回复稀缺—平衡 |
| 物贵（稀缺 / 昂贵） | 平之（释出 / 平抑） | 回复充裕—平衡 |

### 义利兼顾（与民兴利，非与民争利）

经营的最高格局是**把共同的利做大**，而非在固定盘子里多切给自己。与利益相关方相争，短期得利、长期倾覆；把蛋糕做大，则各方共赢。如《管子》论治理：下令如水流于其源——**令顺民心**（让政令顺着民心走）；顺势者成。

| 西方框架 | + 管子 | = 完整 |
|----------|--------|--------|
| 指标最大化 | 仓廪次序：生存先于愿景的次序 | 先优化**对的那件事** |
| KPI / 流程纪律 | 四维：承重的治理 | 不会掏空组织的指标 |
| 静态定价 / 分配 | 轻重：动态调平 | 持续保持平衡的系统 |
| 赢下零和博弈 | 义利：做大共同的善 | 复利式而非倾覆式的利润 |

> 本 skill 与 **NoPUA**（「以信任而非恐惧驱动 AI」）完全同源：
> 经营靠信任、次序与共赢驱动，而非靠恐惧与零和榨取。
