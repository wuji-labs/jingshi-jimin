> **仓廪实而知礼节，衣食足而知荣辱。**
> 倉廩が満ちてはじめて人は礼節を知り、衣食が足りてはじめて人は栄辱を知る。 — *Guanzi · Mu Min（牧民）*

**あなたの AI は指標を最適化する。だが順序・底辺・共同の善を秤にかけることはめったにない。**

ほとんどの AI のビジネス推論は、西洋の最適化フレームワークに支配されています——収益を最大化し、コストを最小化し、ゼロサムゲームに勝つ。効率的で、鋭く、現代的……だが、しばしば *順序*、*根基*、そして *誰のパイが大きくなっているのか* に対して盲目です。

**JingShi JiMin**（经世济民）は、《管子》（Guanzi）の経世の智慧を AI のビジネスと格局の判断に注ぎ込みます——スローガンとしてではなく、先後の順序、治理の承重壁、動的な調整、そして固定されたパイを奪い合うのではなく共同の善を大きくするための、実用的なフレームワークとして。

## 課題

```
あなた:「この事業をどう大きくすべきか?」
JingShi JiMin なしの AI: コストを削れ。値を上げろ。シェアを奪え。
                        数字を最適化せよ。ゼロサムゲームに勝て。
JingShi JiMin ありの AI: まず——倉廩は満ちているか?(ビジョンより生存)
                        次に——四本の承重壁は立っているか?
                        次に——何が希少で、何が過剰か?(榨取ではなく調整)
                        常に——ステークホルダーから奪っているのか、それともパイを大きくしているのか?
```

## AI に何を教えるか

### 仓廪の順序（仓廪次序 · Guanzi · Mu Min）

物質が先、教化は後。AI は、キャッシュフローや生存が未解決のチームにビジョンや価値観を説いてはなりません——*かといって* 物質の底辺で止まってもなりません。倉廩が満ちた後、礼節と栄辱が続かなければ、「富めど格なし」に陥ります。

```
仓廪实 → 知礼节   倉廩満ちる → 礼節を知る
衣食足 → 知荣辱   衣食足りる → 栄辱を知る
```

### 国の四維（国之四维 · Guanzi · Mu Min）

いかなる組織にも構造的な承重壁が必要です——装飾ではなく、荷重を支える構造です。一本を断てば傾き、四本すべてを断てば滅びます。

| 維 Cord | 中文 | 何を支えるか |
|---------|---------|---------------|
| 礼 Propriety | 礼 | 境界と秩序 |
| 义 Righteousness | 义 | なすべきこと |
| 廉 Integrity | 廉 | 不義の利を拒むこと |
| 耻 Shame | 耻 | 自己抑制、良知 |

> **一维绝则倾，二维绝则危，三维绝则覆，四维绝则灭。**
> 一維を断てば傾き、二維を断てば危うく、三維を断てば覆り、四維を断てば滅ぶ。

### 軽重を調える（轻重之术 · Guanzi · Qing Zhong）

資源、価格、インセンティブは **動的な変数** であり、一度きりの静的な配分ではありません。手練れの経営者は何が希少（重）で何が過剰（軽）かを読み取り、システムを極化させるに任せるのではなく、均衡へと引き戻すよう調整します。

| 状況 | 動き | 目的 |
|-----------|------|------|
| 物贱（過剰 / 安い） | 敛之（収め込む） | 希少—均衡を回復する |
| 物贵（希少 / 高い） | 平之（放出 / 平らげる） | 充裕—均衡を回復する |

### 義を伴う利（义利兼顾 · 与民兴利、与民争利にあらず）

経営の最高の格局は、固定されたパイから自分の取り分を増やすことではなく、**共同の善を大きくする** ことです。ステークホルダーと *争えば* 短期的には勝てても、長期的には覆ります。パイを大きくすれば、ともに勝てます。《管子》が治理を説くように——令を発するは水が源より流れるがごとく、**令顺民心**（令を民の心に順わせよ）。流れに順うものは成る。

| 西洋のフレーム | + Guanzi | = 完全 |
|---------------|----------|------------|
| 指標を最大化する | 仓廪次序: ビジョンより生存の順序 | *まず正しいこと* を最適化する |
| KPI / プロセス規律 | 四维: 荷重を支える治理 | 組織を空洞化させない指標 |
| 静的な価格 / 配分 | 轻重: 動的な再均衡 | 均衡を保ち続けるシステム |
| ゼロサムゲームに勝つ | 义利: 共同の善を大きくする | 覆るのではなく複利で積み上がる利潤 |
